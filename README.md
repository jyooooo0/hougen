# 🍒 山形県方言分布ダッシュボード

山形県内35市町村の方言アンケート結果を可視化するインタラクティブなダッシュボードです。

## 機能

- **方言分布マップ**: 地図上で方言の分布を確認できます。
- **詳細分析**: 設問ごとの回答詳細や、市町村別の傾向を分析できます。
- **方言解説**: 「もっけだの」の秘密や、山形の方言が多様な理由を解説します。

## 使用技術

- Python
- Streamlit
- Pandas
- Plotly
- Folium

## ローカルでの実行方法

```bash
pip install -r requirements.txt
streamlit run app.py
```
